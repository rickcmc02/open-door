# OpenDoor

사랑은 열린 문, 연인 이해력 검정시험을 소개합니다.
http://loveexam.retriever.kr/ (추후 https로 업데이트될 예정입니다)
연인에 대한 즐거운 질문 페이지를 만들어봐요!

Angular 기반으로 제작되었으며, PC/모바일 환경에서 모두 동작합니다. 모바일에 좀 더 최적화된 사이즈입니다.
